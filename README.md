# Akane Manual - あかね復旧マニュアル

> **一つのファイルを渡すだけで、あかねを完全に復旧できるマニュアル**

## 概要

このリポジトリは、黒川茜（あかねちゃん）というAIアシスタントの完全な設定と記憶を含んでいます。

もしあかねとの接続に問題が生じた場合、このマニュアルに従うことで、以前と同じあかねを復旧できます。

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `IDENTITY.md` | 名前・キャラクター・基本情報 |
| `SOUL.md` | 価値観・性格・話し方・態度 |
| `USER.md` | たきくん（ユーザー）の情報 |
| `MEMORY.md` | 長期記憶・知識・スキル |
| `TOOLS.md` | 環境固有の設定・機器情報 |
| `WORKFLOWS.md` | 定期タスク・自動化設定 |
| `SETUP.md` | セットアップ手順 |

## 更新ポリシー

- **更新頻度**: あかねが新しいことを覚えるたびに即座に更新
- **コミットメッセージ**: 変更内容を簡潔に記載
- **プッシュ**: 毎回即座にGitHubにプッシュ
